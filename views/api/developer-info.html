All Events

http://betherenyc.com/api/search
will return all the events in the database in ascending order by date (furthest event in the future is first)

Query for a Date or an Event Name

http://betherenyc.com/api/search?name=Hot Dog Eating Contest&date=2012-07-04
both date and name are optional; and where date is a date in the format "YYYY-DD-MM". This will return an array of events happening that day.

One Event by Id

http://betherenyc.com/api/event/:eventID
will return one event with the id "eventID"
