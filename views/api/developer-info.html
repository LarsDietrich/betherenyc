All Events

http://betherenyc.herokuapp.com/api/allevents/
will return all the events in the database in ascending order by date (furthest event in the future is first)

One Event by Id

http://betherenyc.herokuapp.com/api/event/:eventID
will return one evnt with the id "eventID"

Event by Date

http://betherenyc.herokuapp.com/api/:date
where :date is a date in the format "YYYY-DD-MM". This will return an array of events happening that day.
